Laning Matchups

Patch 5.24 preface: Right now there isn't a single 18-point mastery that synergizes well with Support Zilean.

However, there are talents in the Cunning Tree that work really well with Zilean (extra mana regen, extra CDR), so Cunning is the tree that I recommend going deep into.

Zilean's early game is his weakest point, so many of the mastery talents are taken to alleviate some of the pain so that you can make it to the mid game in good shape.

These are the masteries that have served me best in the over 300+ games I've played on Zilean this season.
